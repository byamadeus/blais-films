# Stills

## Drop hero stills and film stills here. Referenced in `src/data/films.js` as `/stills/<filename>`.

## To convert img to webp, run dev first.
drop image.jpg into public/stills/
↓
npm run dev  (script runs, generates image.webp, deletes nothing)
↓
add '/stills/image.webp' to films.js
↓
delete image.jpg if you want
↓
git add + commit  (only .webp files go into the repo)
